
== Concept

This lesson describes what Serialization is and how it can be manipulated to perform tasks that were not the original intent of the developer.

== For this lesson:
* The user *should* have a basic understanding of the Java programming language
* The user will be able to detect insecure deserialization vulnerabilities
* The user will be able to exploit insecure deserialization vulnerabilities
* Exploiting deserialization is slightly different in other programming languages such as PHP or Python, although the
  concepts discussed here apply to any language
