Название урока: Использование непроверяемых почтовых сообщений

Тема для изучения:

Многие сайты позволяют не аутентифицированным пользователям отправлять почтовые сообщения своим "друзьям". Такие сайты представляют из себя хороший инструмент для спамеров, которые получают возможность слать рекламу с использованием почтового сервера компании.

Основные цели и задачи:

Пользователь должен отослать любое почтовое сообщение.
